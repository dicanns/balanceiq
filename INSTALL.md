# Install Instructions — BalanceIQ

## Mac

1. Download `BalanceIQ-x.x.x-universal.dmg`
2. Open the .dmg and drag BalanceIQ to your Applications folder
3. Double-click to launch

**If you see "BalanceIQ is damaged and can't be opened":**

Open Terminal and run:
```
xattr -cr /Applications/BalanceIQ.app
```
Then right-click the app → Open.

This message appears because the app is not yet code-signed with an Apple Developer certificate. It is safe to run.

---

## Windows

1. Download `BalanceIQ-Setup-x.x.x.exe`
2. Double-click the installer and follow the prompts

**If Windows SmartScreen shows "Windows protected your PC":**

Click **More info** → **Run anyway**.

This message appears because the app is not yet code-signed. It is safe to run.
